## Core Philosophy

- **AI Engineer ≠ ML Researcher.** You're not training foundation models or writing papers.
- **Build in public.** A visible trail of real projects on GitHub, demos, and explanatory threads beats any credential.
- **Proof over pedigree.** Companies care about your ability to wire existing models (Claude, GPT, open-source LLMs) into working systems — not theoretical credentials.
- **The model is your tutor.** Use the LLM itself to learn: analogy → smallest working example → exercise → critique.

## How to Use This Curriculum

1. **Follow the stack in order.** Each layer builds on the previous one. Don't skip.
2. **Build the projects.** Reading without building is just entertainment.
3. **Ship publicly.** Maximise your proof surface — GitHub, Twitter threads, demo videos.
4. **Avoid traps:** tutorial hell, waiting to "feel ready," learning in the wrong order, private repos, copying code you can't explain.
