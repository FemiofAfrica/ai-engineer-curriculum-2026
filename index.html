📅 90-Day Plan

Phase 1: Foundations

Days 1–30
  • Python fundamentals (functions, classes, async)
  • SQL & data handling fluency
  • Git, CLI, Linux basics — daily comfort
  • First LLM API calls (OpenAI / Anthropic)
  • Start building in public — your first thread about what you're learning

Phase 2: Build the Proof

Days 31–60
  • Embeddings & vector search — implement semantic search
  • Build Project 1: RAG application on your own data
  • Agent frameworks & tool use — function calling, planning
  • Build Project 2: AI Agent with 3+ tools
  • Write threads explaining your architecture decisions

Phase 3: Ship & Scale

Days 61–90
  • Deployment & MLOps — Docker, cloud deploy, monitoring
  • Build Project 3: deployed full-stack AI product
  • Contribute to open source (small fixes, docs, or a new project)
  • Cold outreach with proof — lead with a working demo
  • Post consistently, specialize in RAG, agents, or AI product engineering
