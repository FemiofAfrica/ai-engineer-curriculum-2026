Build a RAG Application

What it proves: You understand retrieval, chunking, grounding, and hallucination prevention — 70% of AI engineering work.

Load Docs→ Chunk→ Embed→ Index→ Query → Retrieve→ Augment Prompt→ Generate + Cite

Project Spec

Minimum Viable Scope

  1. Take a set of documents (PDFs, Markdown files, or plain text)
  2. Chunk them using a reasonable strategy (semantic or recursive splitting)
  3. Embed chunks and store in a vector DB (Chroma, Pinecone, or pgvector via sqlite-vec)
  4. Build a Q&A interface — CLI or web — that retrieves relevant chunks and generates a grounded answer
  5. Every answer MUST include source citations (which document, which chunk)

Stretch: Hybrid Search

Combine BM25 keyword search with vector search for better results on ambiguous queries.

Stretch: Query Rewriting

Rewrite user questions before retrieval to improve search quality (e.g. expand abbreviations).

Stretch: Streaming

Stream the LLM response token by token instead of waiting for the full answer.

Architecture Skeleton

rag_pipeline.py
import chromadb
from openai import OpenAI
from typing import List

client = OpenAI()
chroma = chromadb.Client()
collection = chroma.get_or_create_collection("my-docs")

def chunk_document(text: str, chunk_size=500, overlap=50):
    # Split text into overlapping chunks
    chunks = []
    for i in range(0, len(text), chunk_size - overlap):
        chunks.append(text[i:i + chunk_size])
    return chunks

def index_document(doc_id: str, text: str):
    chunks = chunk_document(text)
    embeddings = client.embeddings.create(
        model="text-embedding-3-small",
        input=chunks
    ).data
    collection.add(
        ids=[f"{doc_id}-{i}" for i in range(len(chunks))],
        embeddings=[e.embedding for e in embeddings],
        metadatas=[{"doc_id": doc_id, "chunk": i} for i in range(len(chunks))],
        documents=chunks
    )

def retrieve(query: str, top_k=5):
    q_emb = client.embeddings.create(
        model="text-embedding-3-small",
        input=[query]
    ).data[0].embedding
    return collection.query(
        query_embeddings=[q_emb],
        n_results=top_k
    )

def answer(query: str) -> str:
    results = retrieve(query)
    context = "\n\n".join([
        f"[Source: {m['doc_id']}] {d}"
        for d, m in zip(results["documents"][0], results["metadatas"][0])
    ])
    resp = client.chat.completions.create(
        model="gpt-4o-mini",
        messages=[{
            "role": "system",
            "content": f"Answer based on this context. Cite sources.\n\n{context}"
        }, {
            "role": "user",
            "content": query
        }]
    )
    return resp.choices[0].message.content
